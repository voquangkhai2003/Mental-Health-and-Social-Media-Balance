This project performs an in-depth analysis of the Social Media and Mental Health Balance dataset from Kaggle
(https://www.kaggle.com/datasets/ayeshaimran123/social-media-and-mental-health-balance).
The goal is to understand how daily screen time, sleep quality, physical activity, and social media platform preferences are associated with stress levels, happiness, and overall mental well-being.

The workflow includes data cleaning, feature grouping (screen-time tiers, age groups, high-stress users), descriptive statistics, visualization, statistical testing (t-test, ANOVA, Kruskal-Wallis), correlation analysis, and the generation of an automatic PowerPoint report with charts and data tables.
The findings provide valuable insights into behavioral patterns and help identify user segments at higher risk of elevated stress.
